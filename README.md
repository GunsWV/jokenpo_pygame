# 🪨 Pedra, Papel e Tesoura (Jokenpô)

Jogo desenvolvido em Python utilizando a biblioteca Pygame, com interface gráfica e interação via mouse.

---

## 🎮 Sobre o projeto

Este projeto simula o clássico jogo Pedra, Papel e Tesoura, onde o jogador escolhe uma das opções clicando na tela, enquanto o computador faz uma escolha aleatória.

O resultado da rodada é exibido visualmente, junto com o placar atualizado.

---

## 🧠 Objetivo

Praticar conceitos fundamentais de programação, como:

- Lógica de decisão
- Estruturas condicionais
- Manipulação de eventos
- Interface gráfica com Pygame
- Organização de código

---

## 🖥️ Tecnologias utilizadas

- Python 3.11
- Pygame
